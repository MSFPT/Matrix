## Types Run

<br>

### Run on Windows

```
C:\MSFPT\Desktop> cd Matrix
```
```
C:\MSFPT\Desktop\Matrix> matrix.bat
```

<br>

### Run on Linux

```
root@MSFPT:~/Desktop/$ cd Matrix
```
```
root@MSFPT:~/Desktop/Matrix$ chmod +x *
```
```
root@MSFPT:~/Desktop/Matrix$ sudo ./matrix.sh 
```
